The best place ti aquire mods for the witcher 3 is by far NexusMods. I will add a guide which will explain the process of downloading the mods for the witcher 3 in the second column. There is a variety of different mods which you can download for The Witcher 3. The most popular mod recently has been the Ray Tracing mod which does exactly what it sounds like; it adds Ray-Tracing capabilities to The Witcher 3.

The Witcher 3 is definitely gonna be like Skyrim, the game will always be around and users will create new content for the game using mods. From my experience, using mods in this game is A LOT of fun, especially when it comes to the inventory (disables the inventory limit) mod and the poor merchants (gives the merchants more money so that you can actually sell your items) mod.

My advice when it comes to playing this game is adding mods after your first playthrough. I think that you should experience the game the way its meants to be played the very first time and utilise the power of modding on your second playthrough.
